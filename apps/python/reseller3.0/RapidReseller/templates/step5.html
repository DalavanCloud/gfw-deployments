Step 5: Create a user on the customer account

When a Google Apps instance is created, it is instantiated with 0 accounts. Resellers can act on behalf of their resold customers to provision accounts.

Currently, the Cloud Directory API does not currently allow for resellers to act on behalf of the customers. The older Google Apps Provisioning API can be used to act on behalf of a customer.
