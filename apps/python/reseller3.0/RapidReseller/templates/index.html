Rapid Reseller

Rapid Reseller is a quick demonstration of the various API methods available to Google Apps Resellers. The demonstration is written in Python on Google App Engine, but API libraries are available in many other languages, including: Java, C#.Net, PHP, Go, Ruby, and NodeJS.

Source code is freely available for interested parties.
